# Navigation

Classes and methods for navigating within publications.

## Page Navigation

Use [goLeft] and [goRight] for visual page navigation.
Use [skipToNext] and [skipToPrevious] for chapter navigation.

## Locator Navigation

Use [goToLocator] to jump to a specific position.
Use [goByLink] to navigate from table of contents entries.

## Physical Pages

For publications with page lists, use [toPhysicalPageIndex]
to navigate to printed page numbers.

## Decorations

Use [applyDecorations] to add visual markers:
- Highlights
- Bookmarks
- Search results
- Annotations
